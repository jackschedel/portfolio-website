My Roles

Strategy Lead - Solar Gators

My Projects

EndGame2 Chess Engine - Rust

Still actively being developed, I started this project as a way to learn the Rust programming language over the summer (2023). With a focus on multithreading and performance, this UCI-compatible chess engine uses minimax tree-traversal on a variety of custom board analysis algorithms. I used alpha-beta pruning and position hashing to improve performance, and plan to further expand my board analysis algorithms in the future. This project is still actively being expanded upon, and serves as a nice pet project to work on when I feel like doing sime highly algorithmic-focused coding.

Solar Gators Strategy Pit GUI - Typescript

Easily perform on-the-fly regression analysis using live telemetry logged from Sunrider. Developed in only 4 days, the strategy section of the Solar Gators Telemetry website allows users to easily graph data from various statistics over time, trim out bad datapoints (which was at one point crucial when our telemetry data was very noisy), perform regression analysis over a given timespan (which can be set to a custom interval seperate from the plotted points), and extrapolate the regression estimate. This wound up being instrumental to our success in the race allowed me to better manage our driving strategy, resulting in us winning FSGP 2023.

gpt.koaladev.io OpenAI API Client Interface - Typescript

I've been using OpenAI API tools since early 2021, well before the ChatGPT explosion. When GPT3.5 was released, I began to integrate it into my development workflow (although I've been using it a bit less frequently lately as I've been working with more esoteric languages), and with GPT4 access being API-only I suddenly had a personal need for a good API client interface. I thought I found it in BetterChatGPT but I eventually wanted even more. Ultimately, I ended up creating a fork of the project so that I could add my desired tweaks and features. As of writing, my fork is 171 commits ahead of the parent, has had over 220 views on GitHub, and has garnered a number of active users, even including some who've added more tweaks.

Tickr Stock Analysis - NodeJS, PHP, and Oracle SQL
